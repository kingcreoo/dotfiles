# dotfiles
 Dotfiles/config files for my arch & hyprland setup. Using rofi, hyprpaper, waybar.

1. Start off by installing:

- hyprpaper
- waybar
- rofi
- starship shell
- alacritty
- neofetch

2. Then simply copy & paste my config files into your .config directory.

3. Other things that must be installed/configured:

- fonts
    - ttf-material-design-icons-extended
    - FiraMono Nerd Font (https://www.nerdfonts.com/font-downloads)
    - ttf-ms-fonts

- nwg-look for gtk theme
    - i am using catppuccin mocha-dark-standard-sky (https://github.com/catppuccin/catppuccin)

- wlogout & pavucontrol

- Make sure you setup your hyprpaper config file to load whatever wallpaper you want
